  2. display 01

    Display는 웹 페이지의 레이아웃(layout)을 결정하는 요소 중 하나의 속성
    1. 블록(block) div, h1, p, ul, ol, form 요소는 대표적인 요소
    2. 인라인(inline) span, a, img요소는 대표적인 요소
    - display: none(없어짐)
    - display: block(나타남)
    - display: inline(글자처럼 box속성을 없애라,잘 안씀)

  4. flex01
    flex02
    flex03

    Flex는 Element의 크기나 위치를 쉽게 잡아주는 도구
    1. justify-content :가로로 요소들 움직임
    - flex-start: 요소들을 컨테이너의 꼭대기로 정렬
    - flex-end: 요소들을 컨테이너의 바닥으로 정렬
    - center: 요소들을 컨테이너의 세로선 상의 가운데로 정렬
    - space-between: 요소들 사이에 동일한 간격을 둠
    - space-around: 요소들 주위에 동일한 간격을 둠
    ex) justify-content: flex-end;
    2. align-items: 세로로 요소들을 움직임

  8. grid01

    Grid Layout은 웹사이트 디자인의 주요한 요소
    1. Flex는 한 방향 레이아웃 시스템이고 (1차원)
    2. Grid는 두 방향(가로-세로) 레이아웃 시스템 (2차원)
    3. Flex보다 더 복합적인 레이아웃 표현이 가능
    4. Grid(2017년도)-Flex(2018년도)
    5. Safari, Chrome, Firefox, Edge 브라우저 지원
